[![Build & Upload](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.yml/badge.svg)](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.yml)

[![Anaconda Platforms](https://anaconda.org/hep-forge/curlpp/badges/platforms.svg)](https://anaconda.org/hep-forge/curlpp)

Feedstock for [curlpp](https://github.com/jpbarrette/curlpp) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

cURLpp is a C++ wrapper for libcURL

## Install

```bash
conda install -c hep-forge -c conda-forge curlpp
```
