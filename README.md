# curlpp-feedstock

[![Build AMD64](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/curlpp-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [curlpp](https://github.com/jpbarrette/curlpp) — part of [hep-forge](https://anaconda.org/hep-forge).

cURLpp is a C++ wrapper for libcURL

## Install

```bash
conda install -c hep-forge -c conda-forge curlpp
```
